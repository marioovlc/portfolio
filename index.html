Caso de Estudio 01

AedusWeb. Una plataforma unificada.

El ecosistema corporativo demandaba una solución multiplataforma real. Desarrollé AedusWeb utilizando Flutter para garantizar portabilidad absoluta sin sacrificar rendimiento nativo.

  • Procesamiento NLP con LLaMA 3.3 (Groq API).
  • Base de datos relacional altamente optimizada (PostgreSQL).
  • Sincronización en tiempo real.

Caso de Estudio 02

AedusApp. Potencia nativa.

Para el back-office empresarial, el navegador no era suficiente. Construí un cliente nativo en Java 21 y JavaFX (tema AtlantaFX) diseñado para manejar grandes volúmenes de datos con cero latencia de interfaz.

  • Multithreading avanzado para UI no bloqueante.
  • Pooling de conexiones extremas con HikariCP.
  • Atajos de teclado y experiencia Power User.
